## What I'm Working On

### Exploratory Data Analysis and Machine Learning Projects

> *Building intuition for data — one dataset at a time.*

My focus right now is developing a strong foundation: cleaning real-world datasets, understanding distributions, building baseline models, and iterating on them. No shortcuts, just solid fundamentals.

```
Raw Dataset
      │
      ▼
 Data Cleaning ──► EDA + Visualizations
      │                    │
      ▼                    ▼
Feature Engineering    Insight Extraction
      │                    │
      ▼                    ▼

      │
      ▼
```

| Visualization | Communicating insights clearly with Seaborn and Plotly |
